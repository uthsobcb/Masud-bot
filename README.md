# Masud Bot
 A Simple Bot for getting user info by giving user id or mentioning user.
 
 ## Naming Inspiration:
 Masud name came from Masud Rana. Who is fictional Charecter. As a Detective role.

 ## Usage: 
 You have to use ``.spy @username`` or ``.spy userid``
